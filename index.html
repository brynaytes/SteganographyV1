Steganography

download jar

Steganography is a method of encoding data into an image. This is program is meant to hide text inside of an image in an unobtrusive way, but larger applications of the tecniques used here can insert other files and even other images.

Essentially this program takes text and an image, then subtracts the ascii values of each letter from pixels representing this text. The resulting image can then be shared and decoded.
